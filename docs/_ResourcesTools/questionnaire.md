# Find Your Perfect Class

Answer these questions to discover which D&D class and archetype matches your preferred playstyle. The questionnaire focuses on combat preferences, magic usage, roleplay interests, and background origins to provide personalized recommendations from the Varlyn ruleset.

You can skip any question you're unsure about—the recommendation engine will work with whatever answers you provide.
